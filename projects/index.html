Projects

Here are just a few of the projects I have worked on during my time as a software developer:

Nexus Food Delivery

A food delivery app with a focus on speed and reliability and an easy-to-use yet modern user interface. I also wrote a Node.js API that powered the app.

Clockulus

iOS

Apple TV

An Apple TV and iOS fullscreen clock app that provides tons of customization options such as animated background effects, a wide variety of font choices, and the option to choose between a digital and analog clock.

Features:

  • Many different customization options
  • An iAP upgrade to the Pro version which unlocks extra content
  • An analog clock as well as a digital clock with the same level of customization possible
  • A Today extension that allows you to put Clockulus into your Notification Center that is just as customizable as the main app
  • Font picker that allows picking any font registered by iOS/tvOS for maximum customization potential

Flippity Flap

An iOS game that has a focus on quick fun that can be picked up and played for quick sessions.

Features:

  • Full support for Leaderboards and Achievements
  • The entire game, including the UI is created with SpriteKit
  • Addicting never-ending gameplay that is designed to be played in short bursts
  • Consumable iAPs to buy in-game currency

RNG Master

iOS

WatchOS

A random number generator app for iOS, WatchOS, and Apple TV that features 11 forms of RNG.

Features:

  • Siri Shortcuts for activating any of the RNG methods at any time even if the app is not running thanks to Background Execution
  • A Pro version iAP that unlocks extra content
  • Support for Splitview and external displays
  • Viewing History of previous random generations with the ability to export the history as a .CSV file
  • Clean minimalistic design that adheres to Apple’s Human Interface Guidelines

Exchange Rate Master

An app for calculating exchange rates between various currencies for iOS.

Features:

  • The ability to convert between many of the world’s currencies in a simple and easy to use manner
  • Leverages the exchangeratesapi.io API for up-to-date information on exchange rates
  • Simple and easy to use

Playtime Keeper

An app for tracking video game playtime for iOS.

Features:

  • Capable of storing an infinite number of games and playtime
  • Automatic backups of playtime to iCloud so you never lose your data
  • Beautiful graphs to represent your data and visualize it
  • Easy to use interface
